Шаблоны java-классов
====================

В этом разделе представлены шаблоны различных java-классов, каждый из которых используется для описания определенного аспекта модели.

- :ref:`Визуализация клеток <PhysiCell_java_Templates_Visualizer>` - описывает отрисовку клеток разных типов в зависимости от тех или иных условий.
- :ref:`Обновление фенотипа <PhysiCell_java_Templates_UpdatePhenotype>` - описывает изменения, происходящие внутри клетки.
- :ref:`Пользовательское правило <PhysiCell_java_Templates_CustomCellRule>` - описывает пользовательское правило поведения клеток определенного типа.
- :ref:`Начальная расстановка клеток <PhysiCell_java_Templates_InitialCellsArranger>` - описывает позиции клеток в начале симуляции модели.
- :ref:`Клеточный контакт <PhysiCell_java_Templates_Contact>` - описывает контакт клеток друг с другом.
- :ref:`Инициализация <PhysiCell_java_Templates_Instantiator>` - описывает события, происходящие при появлении новой клетки.
- :ref:`Событие <PhysiCell_java_Templates_Event>` - описывает мгновенное изменение, происходящее в модели, под действием какого-либо сигнала.
- :ref:`Подвижность <PhysiCell_java_Templates_Motility>` - описывает изменение движения клеток.
- :ref:`Отчет <PhysiCell_java_Templates_Report>` - описывает создание таблицы отчета по модели.
