Чтобы настроить и запустить симуляцию модели нужно нажать ЛКМ на вкладку **Simulation**.

.. figure:: /images/Physicell/Physicell_simulation/Simulation_tab.png
   :width: 100%
   :alt: Simulation_tab
   :align: center

:raw-html:`<br>`
После этого в правом нижнем углу экрана у вас появится панель для настройки симуляции модели.

.. figure:: /images/Physicell/Physicell_simulation/Simulation_panel.png
   :width: 100%
   :alt: Simulation_panel
   :align: center

:raw-html:`<br>`
На этой вкладке можно выделить несколько областей:

.. figure:: /images/Physicell/Physicell_simulation/Simulation_panel_composition.png
   :width: 100%
   :alt: Simulation_panel_composition
   :align: center

:raw-html:`<br>`

- кнопки для управления процессом симуляции (А),
- вкладки для переключения между настройками параметров симуляции (Engine) и меню добавления переменных для вывода на график (Plots) (Б),
- область редактирования параметров симуляции (Engine) и меню добавления переменных для вывода на график (Plots) (В),
- область вывода логов (Г).

Далее мы подробно разберем каждую из этих областей.
