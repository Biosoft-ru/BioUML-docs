Создание PhysiСell модели
=========================

.. role:: raw-html(raw)
   :format: html

**Physicell** - это платформа для моделирования многокомпонентных систем, в которой каждый объект выступает как отдельный агент.

**Основные ресурсы**:

- Сайт платформы: `https://physicell.org/ <https://physicell.org/>`_,
- Основная статья по PhysiCell: `PhysiCell: An open source physics-based cell simulator for 3-D multicellular systems <https://dx.doi.org/10.1371/journal.pcbi.1005991>`_,
- Статья, в которой описываются правила моделирования поведения клеток: `Human interpretable grammar encodes multicellular systems biology models to democratize virtual cell laboratories <https://www.cell.com/cell/fulltext/S0092-8674(25)00750-0?_returnURL=https%3A%2F%2Flinkinghub.elsevier.com%2Fretrieve%2Fpii%2FS0092867425007500%3Fshowall%3Dtrue>`_.

Далее мы подробно рассмотрим, как создавать PhysiCell модели в BioUML.
